# You can do the following to download and run the project
  ```
  $ git clone https://github.com/Kevinak1992/boot.git
  
  $ cd boot
  
  $ npm install
  
  $ npm start
  ```
  
# You can do the following to perform other operations
  
  * "dist":
  ```
  $ npm dist
  ```
  
  * "start server":
  ```
  $ npm start
  ```
    
  * "karma start":
  ```
  $ npm test
  ```  
  
# You can see more details in the package.json
